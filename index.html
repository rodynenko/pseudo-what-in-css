Pseudo-elements

It is a keyword added to a selector that lets you style a specific part of the selected element(s).

            
h1::first-letter,
h1::after,
h1::selection {}

Pseudo-classes

A pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s).

            
h1:hover,
h1:empty,
h1:target {}
